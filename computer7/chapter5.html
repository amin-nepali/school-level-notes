Scratch Programming – Grade 7

Textbook Exercise Answers

1. Short Answer Questions

a. What is the default sprite in Scratch?
The default sprite is an orange cat.
b. List the different code blocks in Scratch.
The code blocks are Motion, Looks, Sound, Events, Control, Sensing, Operators, Variables, and My Blocks.
c. Write any two features of Scratch programming.
Scratch is easy to use and it uses visual blocks for coding.
d. Which block would you use if you have to repeat any action 10 times?
I would use the “Repeat” block.
e. What is the other name for background in Scratch?
The background is also called a backdrop.
f. List any 4 components of Scratch interface.
The four components are Stage, Sprite List, Blocks Palette, and Script Area.
g. Which block is used if you want to check a condition?
I would use the “If” block.

2. Long Answer Questions

a. Discuss the various features of Scratch.
Scratch has visual block-based coding, is beginner-friendly, lets us make games and animations, has many sprites and backdrops, and it does not have syntax errors.
b. "Scratch is the best programming tool for learners." Explain.
Scratch is best because it is easy to use, helps us think logically and be creative, does not have errors in syntax, and gives instant results on the screen.
c. Define stage area in Scratch.
The stage area is where sprites act and where we see our project happen.
d. What is a sprite in Scratch? Why is it important?
A sprite is a character or object that we control. It is important because it is the main actor in the project.
e. Discuss the use of code area in Scratch.
The code area is where we put blocks together to make the sprite do actions.
f. If you need to change the background of the stage, which option would you choose?
I would click “Choose a Backdrop” and then pick from the library, paint my own, or upload an image.
g. What is the role of a sprite in a project?
Sprites do the actions we program and they can work with the stage, backdrops, and other sprites.
h. How can you move a sprite across the stage?
I can use Motion blocks like “Move steps”, “Go to x y”, or “Glide to” to move the sprite.
i. If you want to make a sprite perform an action many times, which block would you use?
I would use the “Repeat” block for a fixed number of times or the “Forever” block for endless repeating.
j. What is the function of the "When Green Flag Clicked" block?
It starts the program when I click the green flag.
k. How can you add and change the backdrop?
I can click “Choose a Backdrop” in the stage area, then pick from the library, paint, or upload my own. I can also edit it with the Backdrop Editor.
