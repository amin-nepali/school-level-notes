üîç Very Short Answers

a. List out any two spreadsheets software.
Two spreadsheets software are MS-Excel and Google Sheets.
b. What is the extension of Excel?
The extension of Excel is .xlsx.
c. What is cell?
Cell is defined as the intersection of a row and a column in a worksheet.
d. What is Name Box?
Name Box is a small box that displays the address or name of the active cell or selected cell range.
e. What is Formula Bar?
Formula Bar is a bar that displays the contents of the active cell and is used to enter or edit formulas and data.
f. List out any two reference operators used in MS Excel.
Two reference operators are Colon (:) used for range (like A1:A10) and Comma (,) used to combine non-adjacent references.
g. List out any two comparison operators used in MS Excel.
Two comparison operators are Equals to (=) and Greater than (>).
h. What is Fill Icon?
Fill Icon is the small square found at the bottom-right corner of the active cell, used for auto-filling data or formulas.
i. What is meant by merging the cell?
Merging cells is the process of combining two or more adjacent cells into a single larger cell for better alignment or presentation.

üí° Long Answers

a. What is a spreadsheet? Give some examples.
Spreadsheet is defined as a computer application that can store, organize, analyze, and calculate data in a tabular format (rows and columns).

Examples: MS-Excel and Google Sheets.
b. What are the features of spreadsheet?
The features of spreadsheet are:
  • Calculations: Performing complex calculations using formulas and functions.
  • Data Organization: Storing data in an organized grid of rows and columns.
  • Data Analysis: Analyzing data through tools like sorting, filtering, and conditional formatting.
  • Visual Representation: Creating charts and graphs to visualize data trends.
c. What is MS-Excel?
MS-Excel is a popular spreadsheet software developed by Microsoft that is specifically designed for data entry, analysis, financial calculations, and generating reports.
d. What is a worksheet?
Worksheet is defined as the primary page or document used within Excel to store and work with data; it is made up of a grid of rows and columns where all the cells are located.
e. What is a workbook?
Workbook is defined as the entire Excel file that holds a collection of one or more worksheets. Think of the workbook as a physical book, and the worksheets as the pages inside it.
f. What do you mean by Excel Filling?
Excel Filling is the process of copying data or a formula to adjacent cells quickly, usually by dragging the Fill Handle (the small square) to automatically populate a series or replicate a calculation.
g. What is a formula? Write its importance.
Formula is defined as an expression that performs calculations on values in a worksheet.

Importance: It allows users to perform automatic calculations, ensuring that the results are instantly updated whenever the input data changes.
h. What is a function? Write its importance.
Function is defined as a pre-defined or built-in formula in Excel that performs specific calculations, like SUM or AVERAGE.

Importance: It makes complex and frequent calculations simpler and faster to apply, saving the user time from manually writing out a long formula.
i. Mention different types of function used in Excel. Explain them.
Different types of functions used in Excel are:
  • Mathematical/Trigonometric Functions: Used for basic and advanced math operations. Example: SUM (adds numbers).
  • Statistical Functions: Used for analyzing data sets. Example: AVERAGE (finds the mean value).
  • Logical Functions: Used for making decisions or testing conditions. Example: IF (returns one value if a condition is true, and another if false).
  • Date and Time Functions: Used to work with dates and times. Example: TODAY (returns the current date).
