6. Answer the Following Questions

a. What is word processing software? Give two examples.
Word processing software is a computer application used to create, edit, format, and print text documents. It provides tools for typing, formatting text, inserting images, tables, and other elements.
Examples:
  • Microsoft Word
  • Google Docs
b. What is MS-Word?
MS-Word (Microsoft Word) is a word processing software developed by Microsoft. It is part of the Microsoft Office suite and is used to create, edit, format, and print text documents. It offers various formatting options, templates, and tools for professional document creation.
c. Write down some features of MS-Word?
Key features of MS-Word include:
  • Text formatting (font, size, color, style)
  • Page layout and margins
  • Insert images, tables, and charts
  • Spell check and grammar check
  • Templates and themes
  • Headers and footers
  • Track changes and comments
  • Mail merge functionality
d. What is header and footer?
Header: A header is text or information that appears at the top of every page in a document. It typically contains page numbers, document titles, or chapter names.

Footer: A footer is text or information that appears at the bottom of every page in a document. It commonly includes page numbers, dates, author names, or copyright information.
e. List out the components of MS-Word Windows.
Components of MS-Word Window:
  • Title Bar
  • Menu Bar/Ribbon
  • Quick Access Toolbar
  • Document Area/Workspace
  • Scroll Bars (Vertical and Horizontal)
  • Status Bar
  • Ruler
  • View Buttons
f. List out the tabs available in Ribbon
Main tabs in MS-Word Ribbon:
  • Home
  • Insert
  • Design
  • Layout
  • References
  • Mailings
  • Review
  • View
g. What is formatting the text?
Text formatting is the process of changing the appearance and style of text to make it more attractive, readable, and professional. This includes changing font type, size, color, applying bold, italic, underline, alignment, spacing, and other visual enhancements to improve the document's presentation.
h. What are bullet and numbering?
Bullets: Small symbols (like dots, arrows, or custom images) placed before each item in a list to organize information in an unordered format.

Numbering: Sequential numbers (1, 2, 3... or a, b, c... or i, ii, iii...) placed before each item in a list to show order or hierarchy in information.
i. What are superscript and subscript?
Superscript: Text that appears slightly above the normal line of text, commonly used for mathematical expressions (like x²), footnote references, or ordinal numbers (1st, 2nd).

Subscript: Text that appears slightly below the normal line of text, commonly used in chemical formulas (like H₂O) or mathematical expressions.
j. What is alignment in Word? Write its types.
Alignment refers to how text is positioned horizontally within a paragraph or document.

Types of Alignment:
  • Left Align: Text is aligned to the left margin
  • Center Align: Text is centered between left and right margins
  • Right Align: Text is aligned to the right margin
  • Justify: Text is aligned to both left and right margins, creating straight edges on both sides
